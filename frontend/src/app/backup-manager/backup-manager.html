Google Drive — User Account (OAuth 2.0)

วิธีตั้งค่า (ใช้พื้นที่ Gmail 15GB ฟรี)

  1. เข้า Google Cloud Console → สร้าง project (ถ้ายังไม่มี)
  2. เปิด API: Google Drive API ใน "APIs & Services" → "Library"
  3. OAuth consent screen → User Type: External → กรอก app name, support email → Save → ในขั้น "Test users" เพิ่ม Gmail ที่จะใช้ backup
  4. Credentials → Create Credentials → OAuth Client ID → Application type: Web application
  5. กรอก Authorized redirect URIs ตามที่ระบบแสดงด้านล่าง (กด เพื่อ copy)
  6. หลังสร้างเสร็จ → copy Client ID + Client Secret มาวางในฟอร์ม
  7. สร้าง folder ใน Google Drive ของคุณ → copy Folder ID จาก URL: drive.google.com/drive/folders/FOLDER_ID
  8. กด "บันทึก" → กด "Connect Google Drive" → login Gmail + consent → เสร็จ!
