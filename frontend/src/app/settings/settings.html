ล็อคการคีย์ข้อมูล

ล็อคปุ่ม "แก้ไข" และ "เพิ่ม" ในหน้า Dashboard สำหรับ user ทั่วไป เพื่อป้องกันการคีย์ข้อมูลในช่วงที่กำหนด

วัน

0 = ใช้วันที่สิ้นสุดแทน (ถ้า > 0 จะคำนวณวันสิ้นสุดอัตโนมัติจากวันเริ่ม)

เว้นว่าง = ไม่จำกัดวันเริ่มต้น

เว้นว่าง = ไม่จำกัดวันสิ้นสุด (ถ้าตั้งจำนวนวัน > 0 จะคำนวณให้อัตโนมัติ)

ล็อค = toggle เปิดด้วยมือ หรือ วันนี้อยู่ในช่วงวันที่กำหนด (อย่างใดอย่างหนึ่ง) Admin/Super Admin ยังใช้งานได้ตามปกติ
