ตั้งค่าการเข้าสู่ระบบด้วย SSO

เปิด/ปิดปุ่ม ThaID และ ProviderID ในหน้า Login + Register (จะแสดงพร้อมป้าย "รอเปิดใช้" จนกว่าจะเชื่อมต่อบริการจริง)

หมายเหตุ: เปิดสวิตช์เพื่อให้ระบบเปิด OAuth login — ปุ่มจะปรากฏในหน้า Login/Register และ redirect ไปยัง MOPH/DGA จริง (ต้องกรอก config OAuth ด้านล่างก่อน)

ProviderID OAuth Config (MOPH)

ระบบจะ match user ด้วย เลขบัตรประชาชน (SHA-256 hash) — ถ้าไม่พบในระบบ จะปฏิเสธและให้ลงทะเบียนก่อน

ThaID OAuth Config (DGA)

match user ด้วย เลขบัตรประชาชน (SHA-256 hash) — user ใหม่จะถูกปฏิเสธ
