ขั้นตอนการเปรียบเทียบ Structure กับ HDC

1

เปรียบเทียบ Structure

กดปุ่ม "เปรียบเทียบ Structure" เพื่อเชื่อมต่อ HDC และตรวจสอบโครงสร้างตาราง

2

ดูผลเปรียบเทียบ

ดูสรุป: ตรงกัน / ต่างกัน / ไม่มีใน Local / ไม่มีใน HDC → กดขยายดูรายละเอียดแต่ละตาราง

3

เลือกตาราง

ติ๊ก checkbox เลือกตารางที่ต้องการ หรือกดปุ่ม "เลือกทั้งหมด" / "เลือก Missing" / "เลือก Different"

4

ดำเนินการ 2 ทิศทาง

HDC → Local: "สร้าง Local" / "Sync จาก HDC" | Local → HDC: "สร้าง HDC" / "Sync → HDC" (ต้องมีสิทธิ์ write ใน HDC)

5

สร้างฟอร์มจาก HDC (ตัวเลือก)

กดขยายตาราง → กดปุ่ม "สร้างฟอร์มจาก HDC" → ส่ง columns ไป Tab สร้างแบบฟอร์ม
